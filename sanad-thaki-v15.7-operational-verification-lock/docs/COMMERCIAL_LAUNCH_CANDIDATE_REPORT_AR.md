# تقرير نسخة الإطلاق التجارية المرشحة — سند ذكي

## 1. الإصدار
`v14.0.0-commercial-launch-candidate-v1`

## 2. القرار
هذه هي النسخة المختصرة المعتمدة للاختبار الأمني الخارجي ثم الإطلاق المنضبط، مبنية على مسار SaaS Security.

## 3. ما تم إنجازه
- تثبيت نسخة Commercial Launch Candidate.
- تقوية ملف .gitignore.
- إضافة docker-compose.staging.yml.
- تحويل كلمة مرور PostgreSQL إلى متغير بيئة.
- إضافة اختبار مختصر Windows/Linux.
- إضافة release-check.
- إضافة GitHub Actions للفحص.
- إضافة ملف تسليم الفحص الأمني الخارجي.
- إضافة تقرير الإطلاق التجاري المرشح.
- تثبيت منع QR/OpenCV/Payment/OCR الخارجي قبل الإطلاق.

## 4. الأثر الأمني
- تقليل خطر رفع الأسرار.
- منع فتح PostgreSQL على host port.
- استمرار companyId من الجلسة فقط.
- استمرار RLS كخط حماية ثانٍ.
- استمرار RBAC/SOD.
- استمرار منع WhatsApp قبل اعتماد المدير المالي.

## 5. الفحوص المطلوبة
- Smoke Test.
- AWS Staging.
- فاتورة واحدة.
- عزل شركة واحدة.
- Backup/Restore.
- فحص حماية خارجي.
- Retest بعد إصلاح الملاحظات الحرجة.

## 6. المخاطر المتبقية
- تحتاج تشغيل فعلي على AWS Staging.
- تحتاج فحص حماية خارجي مستقل.
- لا تستخدم بيانات عملاء قبل نجاح الفحص.
- لا تحتوي على QR/ZATCA/OpenCV/Payment Gateway حاليًا بناءً على قرار تأجيلها لما بعد الإطلاق.

## 7. قرار القبول
مقبولة كنسخة Launch Candidate فقط، وليست إنتاجًا نهائيًا حتى تنجح بوابات الأمان.
