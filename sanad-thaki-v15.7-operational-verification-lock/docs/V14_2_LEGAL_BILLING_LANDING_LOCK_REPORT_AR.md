# تقرير v14.2.0 — Commercial Legal, Billing & Landing Lock

## القرار
تمت ترقية النسخة التجارية إلى `v14.2.0-commercial-legal-billing-landing-lock` بناءً على اعتماد النسخة v14.1.0 وإضافة متطلبات الباقات، سياسات الاستخدام، الشروط، وصفحة الهبوط.

## ما أضيف
- صفحة داخلية للشروط والسياسات داخل النظام.
- صفحات/وثائق Markdown للشروط والأحكام، الخصوصية، الاستخدام العادل، الاشتراك والدفع، الدعم، واتساب، حفظ وحذف البيانات.
- إظهار الباقات وحدود الاستخدام: الأساسية 99/100، النمو 249/400، الاحترافية 499/1200.
- تثبيت شرط: الفواتير غير المستهلكة تنتهي بنهاية الشهر ولا تدوّر.
- صفحة هبوط خفيفة `/landing.html` للتسويق الأولي بدون تضخيم الواجهة.
- صفحة سياسات عامة `/legal.html` قابلة للمراجعة قبل الإطلاق.

## ما لم يتغير
- لم يتم توسيع المنتج إلى نظام محاسبة.
- لم يتم منح أدمن سند ذكي صلاحيات مالية داخل شركات العملاء.
- لم يتم إضافة مكتبات ثقيلة أو تعقيد معماري جديد.
- بقيت الصلاحيات والعزل من الخلفية.

## حالة النسخة
صالحة للرفع على GitHub و Staging وفحص القبول التجاري والأمني.
غير صالحة لبيانات العملاء الحقيقية قبل الفحص الأمني الخارجي واختبار العزل والنسخ الاحتياطي ومراجعة قانونية نهائية للشروط.
