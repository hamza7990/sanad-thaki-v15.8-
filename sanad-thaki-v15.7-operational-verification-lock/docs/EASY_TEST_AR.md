## الاختبار اليدوي الوحيد بعد التشغيل
1. فاتورة واحدة:
   - Accountant يدخل فاتورة.
   - يرسلها للمراجعة.
   - Finance Manager يعتمدها.
   - يظهر زر WhatsApp بعد الاعتماد فقط.

2. عزل شركة واحدة:
   - مستخدم شركة A لا يرى بيانات شركة B.
