# ملخص السياسات التجارية داخل النسخة

- الباقات: الأساسية 99 / النمو 249 / الاحترافية 499.
- حدود الفواتير: 100 / 400 / 1,200 شهريًا.
- الفواتير غير المستهلكة تنتهي بنهاية الشهر ولا تدوّر.
- أدمن سند ذكي يدير المنصة ولا يدخل في مالية العميل.
- واتساب بعد الاعتماد المالي فقط.
- الدعم الرسمي عبر تذكرة داخل النظام، وواتساب للدعم كمتابعة اختيارية.
- هذه السياسات تحتاج مراجعة قانونية نهائية قبل الإنتاج العام.


## تحديث الباقات v14.2.1

- الأساسية 99: تنظيم فواتير العملاء ومراجعتها فقط، ولا تشمل واتساب أو مطابقة البنك أو التقارير المتقدمة أو التصدير المتقدم أو دعم الأولوية.
- النمو 249: تسريع التحصيل ومتابعة العملاء وإرسال واتساب ومطابقة بنك، مع تقارير تحصيل وتصدير وحدود متوسطة.
- الاحترافية 499: للفواتير الأعلى التي تحتاج تحكمًا ماليًا أوسع وتقارير متقدمة، مع استخدام أعلى ورسائل أكثر ودعم أولوية.

تُفرض هذه الميزات والحدود من الخلفية وقاعدة البيانات وواجهات API، وليست إخفاء أزرار في الواجهة فقط.
