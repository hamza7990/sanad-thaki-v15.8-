# تسليم مختصر للمبرمج/DevOps

المطلوب تشغيل سند ذكي كـ Staging احترافي دون تعديل يدوي على السيرفر.

ممنوع:
- تعديل ملفات مباشرة داخل السيرفر.
- رفع ZIP إلى السيرفر.
- وضع مفاتيح داخل GitHub.
- فتح PostgreSQL للعامة.
- استخدام بيانات عملاء حقيقية قبل الفحص الأمني الخارجي.

المطلوب:
- استخدام `.github/workflows/deploy-staging-ec2.yml`.
- وضع أسرار GitHub المطلوبة.
- تشغيل Deploy Staging to EC2.
- التأكد من `/health`.
- اختبار فاتورة واحدة.
- اختبار عزل شركة واحدة.
- اختبار Backup/Restore.
- تسليم تقرير الفحص الأمني الخارجي.

النسخة: `v14.0.0-commercial-launch-candidate-v1`.
