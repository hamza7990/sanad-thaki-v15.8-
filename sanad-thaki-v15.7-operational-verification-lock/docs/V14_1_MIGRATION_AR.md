
# ترقية v14.1 — لوحة تشغيل المنصة

في نشر Staging نظيف لا تحتاج أمر ترقية لأن `001_schema_rls.sql` يحتوي الجداول والسياسات.

إذا كان السيرفر يعمل على قاعدة بيانات قديمة من v14.0، شغّل أمر الترقية المرفق مرة واحدة فقط بعد رفع النسخة:

```bash
bash ops/ec2/apply-v14-1-migration.sh
```

بعدها يدخل أدمن سند ذكي بالحساب المحدد في `.env`:

```text
PLATFORM_ADMIN_EMAIL
PLATFORM_ADMIN_PASSWORD
```

هذا حساب مشغل المنصة فقط، وليس حساب شركة عميلة، ولا يملك صلاحيات الفواتير أو الاعتماد أو المطابقة أو الإرسال المالي.
