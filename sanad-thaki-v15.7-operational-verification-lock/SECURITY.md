# سياسة الأمن — سند ذكي

## قبل أي بيانات عملاء
ممنوع استخدام بيانات عملاء حقيقية حتى ينجح:
- AWS Staging.
- فحص عزل الشركات.
- فحص Backup/Restore.
- فحص الأسرار.
- الفحص الأمني الخارجي.

## الأسرار
لا ترفع إلى GitHub:
- .env
- مفاتيح OpenAI
- مفاتيح WhatsApp
- AWS keys
- ملفات .pem أو .key
- أي بيانات عملاء

## قاعدة البيانات
- PostgreSQL لا يفتح على الإنترنت.
- المنفذ 5432 ممنوع للعامة.
- companyId يؤخذ من JWT/Session فقط.
- RLS خط حماية ثانٍ.

## عند ملاحظة أمنية حرجة
- أوقف التشغيل التجاري.
- فعّل وضع الصيانة.
- خذ Backup.
- عالج المشكلة.
- أعد الاختبار.
- وثق القرار.
